## amazingLeetcode

任务：在未来一年内搞定LeetCode和剑指Offer，一天3题。这个项目要求长期更新，长期优化。基础知识每个做6道题后，开始将高级知识也加入做题的队列中。

> 加粗部分为特别常见的面试类型。也可以说是编程思想中的重中之重

> 前16个类型，是基础中的基础，是各种编程思想算法的奠基石。

计算机5大基本思想：

- 分治
- 回溯
- 贪心
- 动态规划
- 递归
